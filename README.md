# Node.js API Boilerplate

Código base para o desenvolvimento do backend de projetos em Node.js

### Requisitos de ambiente

- Node e NPM
- Docker e Docker Compose

### Rodando o projeto

`docker-compose up -d`

- Documentação Swagger: http://localhost:4444/swagger
